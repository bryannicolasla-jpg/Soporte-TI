# Menu de Soporte-TI
Este es un Script para pruebas de Soporte desde la terminal de Windows

 [1] Pruebas de Redes---------------- |
 [2] Pruebas de mantenimiento----- |  |
 [3] Salir al CMD estandar         |  |
                                   |  |
                                   |    [1] Realizar PING (Verificar conectividad)
                                   |    [2] Muestra IPCONFIG (Configuracion de red) 
                                   |    [3] Ejecutar NETSTAT (Ver conexiones activas)
                                   |    [4] Realizar TRACERT (Trazar ruta de red)
                                   |    [5] Obtener GETMAC (Direcciones fisicas de red)
                                   |    [6] Manda NSLOOKUP (Info dominio DNS)
                                   |    [7] Ejecutar SpeddTest (Prueba de velocidad)
                                   |    [8] Volver al menu anterior
                                   |    [9] Salir al CMD estandar
                                   |
                                   | 
                                     [1] Reparador de red (Limpieza profunda)
                                     [2] Optimizacion de sistema (optimizador)
                                     [3] Salir

Ejecuta el siguiente menu .bat en tu ordenador, es muy recomendable para el acceso rapido a pruebas en consola que te daran la presicion que deseas en tu labor como tecnico de soporte TI.
